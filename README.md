# Silicon Compiler Docker Image

Use the [Docker](https://en.wikipedia.org/wiki/Docker_(software)) image created on [GitHub](https://github.com/philiprbrenan) by (https://github.com/philiprbrenan/silicon_compiler_docker_image_asic)
to demonstrate placing a previously synthesized [module](https://en.wikipedia.org/wiki/Modular_programming) A multiple times in a [module](https://en.wikipedia.org/wiki/Modular_programming) B using [Silicon Compiler](https://docs.siliconcompiler.com/en/latest/index.html) .
